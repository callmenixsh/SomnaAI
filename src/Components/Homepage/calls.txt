/api/visualize
Purpose: Takes the user's dream text input and returns a visualization or emotional decoding of the dream.
Used in: Portal component (POST request with dream text to get visualized dream data/image).



/api/dreams
Purpose: Fetches a list or archive of recorded dreams with details like title, description, images, tags, and timestamps.
Used in: Gallery component (GET request to load and display dream cards).



/api/emotion-analysis
Purpose:
Provides emotional trend data over time (e.g., positivity values and labels for time series) for charts and insights.
Used in: Patterns component (GET request to fetch emotion data to plot a line chart).
